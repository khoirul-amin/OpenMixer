# USB

Folder ini berisi implementasi USB Device.

OpenMixer menggunakan TinyUSB sebagai USB Stack.

## Features

- USB CDC
- USB Audio Class 2.0
- Descriptor
- Endpoint
- Isochronous Transfer

## Goal

Membuat OpenMixer dikenali sebagai USB Audio Device tanpa driver tambahan.

Supported OS

- Windows
- macOS
- Linux
